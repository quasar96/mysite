Основы Джанго

django-admin startproject mysite

python manage.py startapp shopapp

Откройте файл settings.py в папке mysite/mysite и добавьте 'shopapp' в список установленных приложений

В папке shopapp создаем папку templates(обязательно «templates»). В этой папке templates создаем папку shopapp(название по названию приложения из пункта 6). В этой папке shopapp/templates/shopapp/ создайте файл index.html.

В файле views.py в папке shopapp/ добавьте следующий код: from django.shortcuts import render def index(request): return render(request, 'shopapp/index.html')

Откройте файл urls.py в папке mysite/mysite. Удалите старый код и добавьте следующий код: from django.contrib import admin from django.urls import include, path urlpatterns = [path('admin/', admin.site.urls), path('', include('shopapp.urls')),]

Создайте файл urls.py в папке shopapp/ и добавьте следующий код: from django.urls import path from . import views urlpatterns = [path('', views.index),]

Теперь вы можете запустить свой сайт с помощью команды в терминале: python manage.py runserver

Работа с pip

pip freeze - Выводит список установленных библиотек

pip freeze > requirements.txt - Эта команда создает файл requirements.txt, содержащий все установленные пакеты и их соответствующие версии.

pip install -r requirements.txt - Воссоздает окружение.

Чтобы создать и активировать виртуальную среду, выполните следующие команды:

# Create a virtual environment
python -m venv env_name
# Activate the virtual environment (Windows)
env_name\Scripts\activate
